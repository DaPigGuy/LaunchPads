# LaunchPads
Adds launch pads :)

# Configuration
To change a type of launchpad's block, open the config.yml and change wateverpadtype (launchpadm nlaunchpad, slaunchpad, elaunchpad, & wlaunchpad) and set the current block id to the one you want
